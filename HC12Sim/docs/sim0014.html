Chip Selects

Chip Selects 0-3 appear as RAM memory. Each chip select enables its own RAM segment into the appropriate address range. However a write to a chip select address will cause an entry to be made in the log if enabled on the File Menu. The entry indicates the event name (Chip Select n Write) the time (in E/P-Clocks), address (hexadecimal), and value written as both decimal and hexadecimal:

CS0W,12301,200,129,81
